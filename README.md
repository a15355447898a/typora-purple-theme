## 主题安装步骤

1. 下载本项目中的`purple.css`文件
2. 打开`Typora`,点击`偏好设置`->`打卡主题文件夹`,将弹出`Typora`的主题文件夹
3. 将下载好的`purple.css`文件放到`Typora`的主题文件夹
4. 关闭并重新打开`Typora`,从菜单栏中选择"主题"->"Purple"即可
